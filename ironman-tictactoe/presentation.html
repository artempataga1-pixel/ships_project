02
Обзор

О проекте

🎮

Что это

Браузерная игра «Крестики-нолики» в визуальном стиле Железного Человека. Два игрока на одном экране.

⚙️

Стек

Чистый HTML5 + CSS3 + Vanilla JS. Без фреймворков, без сборщика, без зависимостей. Один файл — браузер — и всё.

✨

Визуал

SVG-анимации ходов, сканлайн-оверлей, пульсирующие индикаторы игроков, модальный оверлей результата.

📱

Адаптивность

CSS Grid с медиазапросом на 500px. На мобильных ячейки сжимаются c 130×130 до 96×96 px автоматически.

## Проект: ironman-tictactoe

Игра «Крестики-нолики» в теме Железного Человека на чистом HTML/CSS/JS. Без фреймворков, зависимостей и сборщика.

### Архитектура

Вся игровая логика находится в одном плоском модуле (`game.js`) с глобальным объектом `STATE`:

```js
STATE = { board: Array(9).fill(null), current: 'X', gameOver: false }
```

Поле — плоский массив из 9 элементов с индексами 0–8. Проверка победы на каждом ходу перебирает `WIN_LINES` (8 константных массивов).

**Поток при клике на ячейку:**
`onCellClick` → запись в `STATE.board` → вставка SVG через `buildXMark(idx)` / `buildOMark(idx)` → `checkWinner()` → `highlightWinners()` или `endGame()` → `updateTurnIndicator()`

### SVG-анимации

Метки динамически формируются как HTML-строки и вставляются через `innerHTML`. Каждый SVG содержит собственный блок `<defs>` с фильтром и градиентом, чьи ID уникальны по индексу ячейки (например `gx3`, `gr7`, `hg7`) — это исключает конфликты ID при нескольких сыгранных ячейках.

- **X (Игрок 1):** Два элемента `<line>` анимируются через `stroke-dashoffset` (99 → 0, длина диагонали ≈ 98.99). Восемь `<polyline>` трещин расходятся от центра с разными `animation-delay`, заданными в CSS через `:nth-child`.
- **O (Игрок 2):** Четыре концентрических `<circle>` — внешнее свечение, лазерное кольцо (`stroke-dashoffset` 201 → 0, длина окружности = 2π×32), выжженная дыра (scale 0→1 с задержкой 0.58s), кольцо раскалённых краёв (бесконечный пульс). Кольцо начинает рисоваться с 12 часов через `transform: rotate(-90deg)` в CSS с `transform-origin: 50px 50px` в координатах SVG.

### CSS-токены дизайна

Все цвета и свечения заданы как CSS-переменные на `:root` — для смены темы достаточно изменить только их. Оверлей со сканлайнами (`.bg-overlay`) использует фиксированный `repeating-linear-gradient` и является чисто декоративным (`pointer-events: none`).

### Важные ограничения

- `overflow: hidden` на `.cell` обрезает SVG-анимации по границе ячейки — не удалять.
- Класс `cell.played` блокирует повторный клик (CSS `cursor: default`); реальная защита — проверка `STATE.board[index]` на не-null.
- Видимость оверлея результата переключается через класс `.visible` и `pointer-events`; задержка 750мс в `endGame` позволяет анимации последнего хода завершиться до появления оверлея.

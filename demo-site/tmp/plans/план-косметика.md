### Задача 4 — Переключатель тёмной/светлой темы ⭐ (самая сложная) — ✅ ВЫПОЛНЕНО

**Нюансы реализации:**
- Использовался `.transitions-enabled` класс на `<html>` с задержкой 50ms после mount — предотвращает flash при первой загрузке
- В Tailwind v4 `@theme inline` — все токены ссылаются через `var()`, переключение через `[data-theme="dark"]` на html-элементе
- Два инстанса ThemeToggle (desktop fixed + mobile navbar) синхронизируются через кастомный `window.dispatchEvent("theme-change")` event — не через Context
- Hydration mismatch предотвращён через `mounted` флаг в ThemeToggle — до mount рендерится нейтральный placeholder без иконки
- Inline скрипт в `<head>` устанавливает data-theme до гидратации — обёрнут в try/catch для Safari private mode
- Navbar изменён с `bg-transparent` на `bg-charcoal/75 backdrop-blur-sm` при scrolled=false — иначе белый текст нечитаем поверх бежевого Hero в светлой теме
- Все тёмные секции (Hero, Partners, Cases, HowWeWork, Testimonials, Clients) адаптированы через CSS переменные `--dark-text-100/80/70/40/10` — исправлено ~20 хардкодных `text-white`
- PartnersTabs — градиент на фото партнёра переведён на `var(--dark-section-bg)`, текст имён/описаний на CSS переменные
- Footer, BottomNav, MobileMenu — намеренно оставлены всегда тёмными (согласно плану: "Navbar/Footer всегда тёмный")
- CaseCard карточки (`bg-charcoal-soft`) — оставлены постоянно тёмными как декоративный элемент, это нормально визуально
- **Коммит:** выполнено в рамках задачи 4

### Задача 4 — Переключатель тёмной/светлой темы ⭐ (самая сложная)

#### Логика тем

| Тема | Белые/светлые блоки | Тёмные блоки (Hero, Testimonials) |
|---|---|---|
| Светлая (по умолчанию) | `#ffffff` | `#dccfbc` + тёмный текст `#3d3326` |
| Тёмная | `#626260` + тёмный текст | `#0a0a0a` + белый текст |

⚠️ **Нюанс:** в светлой теме на фоне `#dccfbc` белый текст не читается → нужно менять `color` внутри тёмных секций.

#### CSS переменные (`globals.css`)
```css
:root, [data-theme="light"] {
  --dark-section-bg:   #dccfbc;
  --dark-section-text: #3d3326;
  --light-section-bg:  #ffffff;
  --light-section-text: #0a0a0a;
}

[data-theme="dark"] {
  --dark-section-bg:   #0a0a0a;
  --dark-section-text: #ffffff;
  --light-section-bg:  #626260;
  --light-section-text: #1a1a1a;
}

/* Плавная анимация для всех элементов */
*, *::before, *::after {
  transition: background-color 1.2s ease, color 1.2s ease, border-color 1.2s ease;
}
```

#### Применение в компонентах
`SectionWrapper.tsx` принимает пропс `dark`:
- `dark={true}` → `style={{ background: 'var(--dark-section-bg)', color: 'var(--dark-section-text)' }}`
- `dark={false}` → `style={{ background: 'var(--light-section-bg)', color: 'var(--light-section-text)' }}`

Блоки с `bg-white` инлайн → заменить на `bg-[var(--light-section-bg)]`

#### Новые файлы
- `src/hooks/useTheme.ts` — хук: localStorage + `document.documentElement.dataset.theme`
- `src/components/ui/ThemeToggle.tsx` — кнопка: `w-10 h-10 rounded-full border border-gold bg-charcoal`, иконки `Sun`/`Moon` из Lucide

#### Размещение кнопки
- **Desktop:** `fixed bottom-6 left-6 z-50` в `layout.tsx`
- **Mobile:** в `Navbar.tsx` рядом с иконкой телефона

#### Исключения (не реагируют на тему)
- Navbar → всегда тёмный
- Footer → всегда тёмный
- Логотип → всегда белый
